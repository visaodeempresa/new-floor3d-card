## Milestones Three.js

| Milestone | URL |
|-----------|-----|
| Listagem de todos | https://github.com/mrdoob/three.js/milestones?state=closed |
| Exemplo (milestone 49) | https://github.com/mrdoob/three.js/milestone/49?closed=1 |

**Nota:** Os milestones numerados NÃO correspondem diretamente ao número do release. O agente executor deve:
1. Acessar https://github.com/mrdoob/three.js/milestones?state=closed
2. Localizar pelo título o milestone do release sendo migrado
3. Verificar issues fechadas naquele milestone para breaking changes não documentadas no guide
